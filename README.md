# Highway-Networks

Highway Networks
Rupesh Kumar Srivastava, Klaus Greff, Jürgen Schmidhuber

https://arxiv.org/abs/1505.00387


This code contains a MNIST training example using Highway Networks and PyTorch.

- n_layers 2: 94.280%
